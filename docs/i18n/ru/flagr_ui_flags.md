# Управление флагами

Список флагов — это главный экран. Отсюда вы создаёте флаги, находите существующие и восстанавливаете удалённые.

![Список флагов](images/ru/flags-list.png)

## Создание флага

В верхней части списка введите короткое описание в поле «Опишите новый флаг», затем воспользуйтесь зелёной кнопкой:

- «Создать флаг» — создаёт пустой флаг только с вашим описанием. Варианты и сегменты вы добавляете сами на следующем экране.
- «Создать простой булев флаг» (в выпадающем списке кнопки) — создаёт готовый к работе флаг с переключателем вкл/выкл: у него сразу есть варианты `on` / `off` и уже настроенный сегмент, так что его можно включить немедленно.

Кнопка неактивна, пока вы не ввели описание. После создания Flagr открывает страницу нового флага и показывает уведомление «Флаг создан».

!> Новый флаг по умолчанию **выключен** и не возвращает ни одного варианта, пока вы его не включите. Сначала настройте флаг, затем включите его. См. [Как работает оценка](flagr_evaluation).

## Поиск флага

Поле **поиска** фильтрует список по мере ввода (результаты обновляются после короткой паузы).

- Нажмите <kbd>/</kbd> в любом месте страницы, чтобы перейти к полю поиска.
- Поиск ведётся по **ID**, **ключу**, **описанию** и **тегам** флага — без учёта регистра.
- Разделяйте запросы **запятой**, чтобы требовать совпадения сразу по *всем* из них. `checkout, beta` находит флаги, в которых есть и «checkout», *и* «beta».
- Чтобы очистить поиск, нажмите **×**.

Счётчик под полем показывает, сколько флагов найдено (`12 флагов`), а во время активного поиска — `из 200`. Если ничего не нашлось, вы увидите **Нет флагов по запросу**; на пустом экземпляре отображается **Пока нет флагов**.

## Таблица

| Столбец | Примечания |
|--------|-------|
| **ID флага** | Числовой ID. Нажмите на заголовок, чтобы отсортировать (по умолчанию — сначала новые). |
| **Описание** | Произвольное описание. |
| **Теги** | Чипы тегов, если они есть. |
| **Кто изменил** | Кто последним изменил флаг. Поддерживает сортировку. |
| **Изменено (UTC)** | Время последнего изменения в UTC. Наведите курсор, чтобы увидеть относительное время («3 часа назад»). Поддерживает сортировку. |
| **Включён** | Индикатор статуса `вкл` / `выкл`. Используйте фильтр столбца, чтобы показать только включённые или только выключенные флаги. |
| **Действие** | Иконка ↗ открывает флаг в новой вкладке. |

- **Нажмите на строку**, чтобы открыть соответствующий флаг.
- **Нажмите на строку с зажатой Cmd/Ctrl** (или нажмите иконку ↗), чтобы открыть его в новой вкладке браузера.
- На узких экранах второстепенные столбцы (Теги, Кто изменил, Изменено) скрываются, чтобы ID, описание и статус оставались читаемыми.

Если флагов больше 50, внизу появляются элементы постраничной навигации.

## Удалённые флаги и восстановление

Удаление флага — это **мягкое удаление**: флаг скрывается, но не уничтожается. Внизу списка раскройте раздел «Удалённые флаги» (он загружается по запросу), чтобы увидеть их.

Нажмите «Восстановить» в любой строке и подтвердите действие — флаг вернётся в активный список ровно в том виде, в каком был. О том, как работает удаление, см. [Редактирование флага → Удаление](flagr_ui_editor).
