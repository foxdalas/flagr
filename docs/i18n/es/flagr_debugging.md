# Consola de depuración de Flagr

Puedes depurar la evaluación de Flagr usando la consola de depuración.
La consola de depuración es un componente del frontend que envuelve la API de evaluación.
Está integrada dentro de cada página de flag y ayuda a probar y experimentar con la configuración del flag
usando entidades.
